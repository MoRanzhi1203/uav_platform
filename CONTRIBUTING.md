# 贡献指南 (Contributing)

感谢您对本项目的关注！我们欢迎任何形式的贡献，包括但不限于提交 Issue、提出新功能建议、修复 Bug 或改进文档。

## 🛠️ 开发流程

1. **Fork 本仓库**：点击页面右上角的 "Fork" 按钮。
2. **克隆到本地**：
   ```bash
   git clone https://github.com/your-username/uav_platform.git
   ```
3. **创建特性分支**：
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **提交代码**：请确保您的代码符合项目的编码规范，并附带必要的注释。
5. **推送分支**：
   ```bash
   git push origin feature/your-feature-name
   ```
6. **开启 Pull Request**：在 GitHub 上提交合并请求，并详细说明您的改动。

## 📝 提交规范

- **标题**：简明扼要地描述改动内容（如：`feat: 新增地块自动切分功能`）。
- **描述**：详细说明改动的背景、实现方式及测试情况。
- **标签**：根据改动类型打上相应的标签（如：`bug`, `enhancement`, `documentation`）。

## 🐞 问题反馈

如果您在使用过程中发现任何问题，请通过 [GitHub Issues](https://github.com/your-username/uav_platform/issues) 提交反馈，并尽可能提供以下信息：
- 操作系统及浏览器版本
- 复现步骤
- 错误日志或截图
