# 重庆山地无人机群林农协同作业与风险管控系统

## 📖 项目简介

本项目是一个基于 Django 和 Leaflet 的无人机林农协同作业与风险管控系统。核心提供了一个专业、高精度的地块创建与编辑工作台，支持山地复杂地形下的无人机作业区域规划、属性管理以及多图层叠加分析。

系统结合了 GIS 技术，实现了精细到 10m 级别的像素级地块绘制，并集成了重庆市行政区划边界、地形等高线等丰富的地理信息数据，为林业巡检、农业植保等无人机作业提供可靠的数字化底座。

## ✨ 核心功能特性

### 🗺️ 专业地块编辑器
- **像素级连续绘制**：支持 10m×10m 精度网格的连续画笔绘制，支持自定义画笔大小（NxN）与形状（方形/圆形）。
- **智能橡皮擦**：提供“整块擦除”与“画笔擦除”双模式，灵活修正绘制区域。
- **多边形合并与布尔运算**：支持对绘制的同类地块进行自动合并，或执行高级的布尔减法操作。
- **纯净交互体验**：深度定制并拦截了浏览器默认鼠标手势与右键菜单，提供类似专业设计软件的无干扰绘制体验。

### 📑 PS 式图层与数据管理
- **图层独立控制**：支持对业务地块、参考网格（10m/1000m）、行政边界进行独立的显示/隐藏、锁定与颜色自定义。
- **属性联动**：地块大类（林区/农田/水域等）与子类别智能联动，支持自定义拓展。
- **自动对齐**：加载时自动计算并平移地图视口，确保目标区域始终居中显示。

## 🚀 快速开始

### 1. 环境准备
确保您的系统已安装以下环境：
- Python 3.10+
- pip (Python 包管理器)

### 2. 安装依赖
克隆项目到本地后，在项目根目录执行：
```bash
pip install -r requirements.txt
```

### 3. 数据库迁移
初始化 SQLite 数据库并应用迁移：
```bash
python manage.py makemigrations
python manage.py migrate
```

### 4. 启动开发服务器
```bash
python manage.py runserver
```
启动成功后，在浏览器中访问 `http://127.0.0.1:8000/` 即可进入系统。访问 `http://127.0.0.1:8000/terrain/editor/` 可直接进入地块编辑工作台。
