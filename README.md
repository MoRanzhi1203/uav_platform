# 无人机林农协同系统 - 地块绘制工作台 (Trae CN 标准项目)

本项目是基于 Trae CN 开发的无人机林农协同系统地块绘制工作台，重点实现了高精度的 10m 连续像素画笔绘制、多尺度参考网格、PS 式图层管理以及行政区划边界联动功能。

## 项目目录结构说明
- **instructions/**: 包含所有 Trae CN 助理指令模板。
- **config/**: 核心地图初始化与绘制逻辑配置。
  - `map_init.js`: 地图初始化与底图配置。
  - `terrain_editor.js`: 绘制逻辑、网格生成、图层管理核心。
- **data/**: 地理信息资源文件。
  - `static/shp/`: 重庆行政区划 SHP 数据。
  - `static/geojson/`: 业务参考数据。
- **scripts/**: 辅助处理脚本。
- **templates/**: 页面 HTML 模板。

## 核心功能特性
1. **10米连续像素画笔绘制**: 支持鼠标拖拽连续生成对齐的 10m x 10m 网格，自动合并同类地块。
2. **多尺度参考网格**: 独立控制 10m (绘制级) 和 1000m (中尺度) 参考线。
3. **PS 式图层管理**: 每一个用户绘制地块、参考网格、行政边界均可独立显示/隐藏、锁定/解锁。
4. **行政边界联动**: 卫星底图模式下可一键加载并显示重庆各区县边界（SHP 数据）。
5. **橡皮擦双模式**: 支持“整块擦除”与基于笔刷的“连续画笔擦除”。
6. **地图平移解耦**: 新增独立平移按钮，支持在不干扰绘制的情况下自由浏览地图。

## 快速开始
1. 确保已安装 Python 环境（推荐 3.12+）。
2. 安装依赖：`pip install -r requirements.txt`。
3. 执行数据库迁移：`python manage.py migrate`。
4. 启动项目：`python manage.py runserver`。
5. 访问：`http://127.0.0.1:8000/` 进入地块编辑页面。
