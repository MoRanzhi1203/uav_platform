# 无人机林农协同系统 - 地块绘制工作台功能更改总结 (Changelog)

## 1. 核心绘制功能
- **10米连续像素画笔绘制**: 
  - 实现了基于 10m x 10m 网格的像素级连续绘制功能。
  - 支持鼠标按下启动、拖动连续生成网格、松开结束绘制。
  - 笔刷大小可自定义（1~N 网格），自动对齐网格并避免重叠。
  - **自动合并**: 绘制结束时，自动将相邻且同类型（类型、风险、子类型一致）的网格合并为单一 GeoJSON 要素。

## 2. 参考网格系统
- **1000米参考网格**: 
  - 添加了中尺度参考网格（1km x 1km），作为大范围规划的辅助参考。
  - 采用淡灰色虚线样式，支持独立显示/隐藏。
- **取消 100km 参考线**: 
  - 根据需求移除大尺度 100km 参考线，精简视觉干扰。

## 3. 图层管理 (PS 式)
- **多图层分离管理**: 
  - 用户绘制地块、1000米参考网格、行政边界图层完全独立。
  - **功能支持**: 每个图层均支持显示/隐藏、锁定/解锁、顺序调整以及删除操作。
- **纯净工作区**: 
  - 初始状态仅显示用户手绘地块和底图。
  - **移除冗余**: 彻底取消了历史地块图层（默认为空）和所有测试样例小样，提高响应速度。

## 4. 橡皮擦功能优化
- **双模式切换**: 
  - **模式1 (整块擦除)**: 点击地块任意位置即可删除整个选中的地块。
  - **模式2 (画笔擦除)**: 采用与画笔相同的连续交互逻辑，按笔刷大小擦除地块的局部网格区域。
- **实时预览**: 擦除时显示红色预览区域，支持撤销/重做操作。

## 5. 底图与行政边界优化
- **标准底图标签**: 
  - 切换底图数据源，确保在缩小比例尺时显示清晰的中文地名和行政区划标签。
- **卫星底图与 SHP 边界**: 
  - 支持加载本地 `shp/chongqing-260411-free.shp` 数据。
  - 在卫星模式下可独立开启“行政区划边界”图层，显示重庆各区县边界红线。
- **范围限制**: 地图视野严格限制在重庆行政区范围内，防止超出瓦片覆盖区。

## 6. 地图视界操作
- **独立平移按钮**: 
  - 新增“平移/拖动”工具按钮。
  - 启用后支持鼠标拖拽平移地图，与绘制/擦除逻辑完全解耦。
- **缩放支持**: 支持平滑的缩放操作，不影响当前绘制图层的稳定性。

## 7. 性能优化
- **分块渲染**: 采用 Leaflet Canvas 渲染器对绘制图层进行分块处理。
- **底图解耦**: 底图加载与地块绘制层独立运行，显著提升了大尺度、复杂地块下的交互响应速度。
